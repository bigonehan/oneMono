# AGENTS.md

## 패키지 범위
- `packages/ui`는 모노레포에서 재사용하는 UI 패키지들을 모아두는 공간이다.
- 새 UI 컴포넌트나 UI 관련 라이브러리 래퍼를 만들기 전에 이 디렉토리 하위의 기존 패키지를 먼저 확인한다.
- 기존 UI 패키지로 표현할 수 있으면 새 패키지를 만들지 말고 기존 패키지를 확장한다.

## 버전 관리
- 기능 추가 또는 기능 개선이 있을 때 변경된 내부 UI 패키지의 `package.json` 버전을 `0.0.1`씩 올린다.
- 별도 지시가 없으면 patch 단위 증가를 사용한다.
