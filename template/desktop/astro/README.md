# template-desktop-astro

Desktop template using Astro + Tauri + SQLite.

## Features
- `@ui/shadcn` list/header/footer UI
- `@features/editor` rich text editor modal
- `@domain/user`, `@domain/article` model concepts
- SQLite persistence via `@tauri-apps/plugin-sql` in desktop runtime

## Scripts
- `bun run dev`
- `bun run tauri:dev`
- `bun run build`
- `bun run check-types`
- `bun run test:e2e`
